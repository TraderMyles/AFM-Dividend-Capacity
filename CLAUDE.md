# CLAUDE.md — Dow 30 Shareholder Analysis

## Project Overview

A static historical financial analysis tool that calculates and visualises
Total Shareholder Return (TSR) and Dividend Capacity for all 30 Dow Jones
Industrial Average companies. Built as a research tool for educational
purposes, rooted in ACCA Advanced Financial Management methodology.

Data is static, pulled once from the Financial Modeling Prep (FMP) API
for fiscal year 2025. The app displays a clear "as of" date so users
understand the data vintage. It is not a live dashboard.

Deployed on Streamlit Cloud. Clean and minimal. No unnecessary UI elements.

---

## Stack

- **Data**: Static JSON files generated by a one-time FMP API pull
- **App**: Streamlit (Python)
- **Deployment**: Streamlit Cloud
- **No database. No ETL pipeline. No live API calls in the app itself.**

## Data

### Source
Financial Modeling Prep (FMP) API — free tier.
Pulled once. Stored as static JSON in `/data/`.

### Fiscal Year
All data is for each company's most recently completed fiscal year
as reported in their annual filing (10-K or equivalent).
This means fiscal year end dates vary across companies.

### Data As Of
Each company record includes a `data_as_of` field showing the fiscal
year end date the data represents. The app displays this clearly.
Do not claim the data is current or real-time anywhere in the UI.

### Companies
All 30 current Dow Jones Industrial Average constituents.
Tickers and names must match the official DJIA constituent list.

## Formulas

### Total Shareholder Return (TSR)

```
TSR = (Closing Price - Opening Price + Dividends Per Share) / Opening Price
```

- **Opening Price**: Share price on the first trading day of the company's fiscal year
- **Closing Price**: Share price on the last trading day of the company's fiscal year
- **Dividends Per Share**: Total dividends declared per share during the fiscal year
- Result expressed as a percentage, rounded to 2 decimal places

### Dividend Capacity (ACCA AFM Method)

```
Dividend Capacity =
  Operating Cash Flow
  - Tax Paid
  - Finance Charges (interest paid, if separately disclosed)
  - Preference Dividends (if applicable)
  - Debt Repayment
  - Share Repurchases
  - Capital Expenditure (investment in assets)
  + Depreciation & Amortisation (non-cash add-back)
  + Proceeds from New Debt Issuance
  + Proceeds from New Share Issuance
```

- All figures sourced from the cash flow statement
- Where interest paid is not separately disclosed, note this as a limitation
- Result in millions USD (or native reporting currency), rounded to nearest million

### Adjusted Dividend Capacity (ex-buybacks)

```
Adjusted Dividend Capacity = Dividend Capacity + Share Repurchases
```

This removes buybacks from the calculation to reveal capacity
available for traditional dividend payments. Shown alongside
standard dividend capacity in the app for comparison.

## Streamlit App

### Structure
- Single page app with sections, not multi-page
- Sidebar for filters and company selection
- Main area for charts and tables

### Sections (in order)
1. **Header** — title, one-line description, data as of date
2. **Overview** — summary stats across all 30 companies
3. **TSR Comparison** — bar chart of TSR for all 30 companies, sorted descending
4. **Dividend Capacity Comparison** — bar chart showing standard and adjusted capacity side by side
5. **Company Deep Dive** — select a company from sidebar, show full breakdown of inputs and outputs with brief interpretation

### Design Rules
- Use Streamlit's default theme — do not add custom CSS unless absolutely necessary
- No emojis in headers or labels
- No marketing language
- All charts via Plotly (not matplotlib)
- Tables via st.dataframe — keep them clean and sortable
- Numbers formatted with commas, $ prefix where applicable, % suffix for TSR
- Negative numbers shown in red in tables
- Every chart has a clear title and labelled axes

### Data As Of Notice
Display this prominently near the top of the app:

> "Data sourced from company annual reports via Financial Modeling Prep.
> Figures represent each company's most recently completed fiscal year.
> This is a static historical analysis, not live data."

## Fetch Script

`scripts/fetch_data.py` is a standalone script run locally, not in the app.

It must:
- Pull cash flow statement data from FMP for all 30 tickers
- Pull historical share price for fiscal year open and close dates
- Pull dividends per share for the fiscal year
- Transform into the schema defined below
- Write to `/data/tsr.json` and `/data/dividend_capacity.json`
- Print a summary of any missing or null values so they can be reviewed

It must not:
- Be imported by the Streamlit app
- Make assumptions about missing data — flag it, don't fill it

## Constraints — Do Not Violate

- Do not add a live API call anywhere in the Streamlit app
- Do not add a database connection
- Do not change the formulas without being explicitly told to
- Do not add authentication or login screens
- Do not add a "calculator" feature where users input their own numbers
- Do not use matplotlib — use Plotly only
- Do not store the FMP API key anywhere in the codebase —
  use a .env file locally and Streamlit secrets for deployment
- Do not rename the main app file — it must be `streamlit_app.py`
- Do not split into multiple Streamlit pages

---

## Known Limitations to Surface in the App

- Finance charges (interest paid) are not separately disclosed by all companies
  in their cash flow statements — noted per company where applicable
- Fiscal year end dates vary across the 30 companies
- Share repurchases are included in the standard dividend capacity formula
  per ACCA methodology — adjusted figure removes this for comparison
- TSR uses fiscal year open/close prices, not calendar year

## Educational Context

This analysis is based on ACCA Advanced Financial Management (AFM / P4)
